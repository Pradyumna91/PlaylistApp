========================================================================
    APPLICATION : ConsoleApplication2 Project Overview
========================================================================

The entire functionality of the application is performed mainly by the following 2 classes:

1. Playlist - This class stores and works with the playlist. All the user commands (create, insert, etc) are implemented in this class.
2. UserInput - This class is used to handle the commands the user passes to the program in the command line. The class is responsible for validating and parsing the user input.

The file ConsoleApplication2.cpp contains the main method which only accept the user input and performs appropriate functions.
